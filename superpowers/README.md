> 84,751 Stars 的 AI 开发技能框架

## 简介

Superpowers 是一个完整的 AI 软件开发工作流框架，基于可组合的"技能"系统。

## 核心理念

1. **先问做什么** - 不直接写代码，先理解目标
2. **自动设计** - 分块展示让你审核
3. **TDD** - 测试驱动开发
4. **子Agent分工** - 多Agent协作
5. **自动触发** - 不需要特殊操作

## 与OpenClaw的对比

| 方面 | Superpowers | OpenClaw |
|------|-------------|-----------|
| 技能格式 | Markdown | SKILL.md |
| 触发方式 | 自动/关键词 | 关键词 |
| 目标 | Claude Code/Cursor | OpenClaw |

## 可借鉴的点

1. **TDD流程** - 编写测试→失败→写代码→通过
2. **调试方法论** - 系统化调试步骤
3. **代码审查** - 请求/接收审查的流程
4. **计划编写** - 清晰的可执行计划
5. **子Agent协作** - 多Agent分工模式
